# 1. Brainstorming & Ideation

## Project Title

Emotion Detection & Learning Support Engine

---

## Problem Statement

Students often struggle with stress, confusion, frustration, and other emotions while learning. Traditional learning platforms provide the same content to everyone and do not consider the learner's emotional state.

There is a need for an intelligent system that can detect emotions from text and provide personalized learning support.

---

## Project Objectives

- Detect user emotions from text input.
- Compare BiLSTM and BERT models.
- Generate personalized learning guidance using Gemini AI.
- Store prediction history.
- Provide interactive analytics and visualizations.
- Build an easy-to-use Streamlit application.

---

## Motivation

The project aims to improve the online learning experience by combining Natural Language Processing, Deep Learning, and Generative AI.

By understanding user emotions, the system can provide more effective support and recommendations.

---

## Key Features

- Emotion detection using BiLSTM.
- Emotion detection using BERT.
- AI-powered guidance generation.
- Analytics dashboard.
- CSV export functionality.
- Prediction history tracking.

---

## Expected Outcome

An intelligent learning assistant capable of understanding emotions and helping students with personalized suggestions and guidance.
