# 4. Project Planning Phase

## Project Timeline

The project was developed in multiple phases, including planning, model development, testing, and deployment.

---

## Development Stages

### Stage 1: Problem Identification

- Identify the limitations of traditional learning systems.
- Define project objectives.
- Select suitable datasets.

---

### Stage 2: Data Collection

Datasets used:

- GoEmotions Dataset
- ISEAR Dataset
- Empathetic Dialogues Dataset

---

### Stage 3: Data Preprocessing

Tasks performed:

- Remove null values.
- Clean text data.
- Tokenize text.
- Encode labels.
- Pad sequences.

---

### Stage 4: Model Development

Models implemented:

- BiLSTM Model
- BERT Model

Objectives:

- Compare performance.
- Improve emotion classification.

---

### Stage 5: AI Integration

- Integrate Google Gemini API.
- Generate personalized learning guidance.
- Provide emotion-based recommendations.

---

### Stage 6: Frontend Development

Framework used:

- Streamlit

Features:

- User input form
- Prediction dashboard
- Interactive visualizations
- History tracking
- CSV export

---

### Stage 7: Testing and Validation

- Verify model predictions.
- Test UI functionality.
- Validate Gemini responses.
- Ensure system stability.

---

### Stage 8: Deployment Preparation

- Organize project structure.
- Upload source code to GitHub.
- Prepare documentation.
