# 8. Project Demonstration

## Project Title

Emotion Detection & Learning Support Engine

---

## Project Description

Emotion Detection & Learning Support Engine is an AI-powered application that detects emotions from user text and provides personalized learning guidance using deep learning and generative AI.

The system combines BiLSTM, BERT, and Google Gemini to improve the learning experience and support students based on their emotional state.

---

## Demonstration Features

### Emotion Detection

- Detects emotions from user input.
- Supports multiple emotion categories.
- Compares BiLSTM and BERT predictions.

---

### AI Learning Guidance

- Generates personalized study suggestions.
- Provides motivational feedback.
- Uses Google Gemini API.

---

### Analytics Dashboard

- Emotion distribution charts.
- Prediction history.
- Confidence visualization.
- CSV export.
